# SkillMatcher

**SkillMatcher** is a smart job-matching platform that allows users to upload their resumes, automatically extracts their skills, and matches them to relevant job opportunities.

Built with **Python** and **Tkinter**, SkillMatcher provides a clean, tabbed user interface for uploading resumes, browsing jobs, messaging recruiters, and managing user profiles.

---

## ✨ Features

- **Upload Resume (PDF or Word)**  
  Upload your resume in .pdf or .docx format for skill extraction.

- **Automatic Skill Extraction**  
  Parses your resume and identifies key skills to match with job listings.

- **Job Matching Engine**  
  Matches your extracted skills to available job opportunities based on keyword analysis.

- **Messaging System**  
  Send and receive messages with recruiters or hiring managers.

- **User Profile**  
  View your extracted skills and personal information in the Profile tab.

- **Simple, Modern Interface**  
  Built with Tkinter's Notebook tabs for easy navigation (Home, Jobs, Messages, Profile).

---

## 🛠 Technologies Used

- Python 3.x
- Tkinter (GUI library)
- PyPDF2 (for PDF parsing)
- python-docx (for Word file parsing)
